# Snake Game

A classic Snake game built with Flutter.

This repository contains a small demo Flutter app implementing the Snake game.

## What's included

- A Flutter app (Dart) implementing the game loop, input handling, and simple UI.
- Instructions to run and iterate locally.

## Quick start

1. Install Flutter: https://docs.flutter.dev/get-started/install
2. Clone the repo:

   git clone https://github.com/Abdelillah-CHADLI/Snake-Game.git
   cd "Snake-Game"

3. Get packages:

   flutter pub get

4. Run on an emulator or device:

   flutter run

## Development notes

- This project targets stable Flutter. If you prefer, run with `flutter channel stable`.
- Add your game code under `lib/` (e.g., `lib/main.dart`) and commit changes.
